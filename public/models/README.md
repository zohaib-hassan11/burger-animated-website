# 3D Models Folder

Place the client's 3D model here as `device.glb`.

**Recommended format:** `.glb` with **Draco compression** (small file size, fast load).

If you receive `.fbx`, `.obj`, or `.blend`:
1. Open in [Blender](https://www.blender.org/) (free)
2. File → Export → glTF 2.0 (.glb)
3. Tick **Compression** in the export dialog
4. Save as `device.glb` in this folder
